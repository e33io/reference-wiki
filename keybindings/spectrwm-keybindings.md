# Spectrwm Keybindings

```
Modifier keys:

Mod4      Windows (Super) key
Mod1      Alt key
Control   Control key
Shift     Shift key
Return    Enter key
space     Space bar
Left      Left arrow key
Right     Right arrow key
Up        Up arrow key
Down      Down arrow key
Tab       Tab key
equal     Equal key
minus     Minus key
Print     PrtSc (Print Screen) key
button1   Left mouse button
button3   Right mouse button
```
```
Custom keybindings:   MOD = Mod4

MOD+d                 Program launcher (app menu)
MOD+Return            Launch terminal
MOD+Shift+Return      Launch file manager
MOD+b                 Launch web browser
MOD+1                 Switch to workspace 1
MOD+2                 Switch to workspace 2
MOD+3                 Switch to workspace 3
MOD+4                 Switch to workspace 4
MOD+5                 Switch to workspace 5
MOD+6                 Switch to workspace 6
MOD+7                 Switch to workspace 7
MOD+8                 Switch to workspace 8
MOD+9                 Switch to workspace 9
Control+Mod1+Right    Switch to next workspace
Control+Mod1+Left     Switch to previous workspace
MOD+u                 Switch to workspace with urgent window
MOD+Shift+1           Move active window to workspace 1
MOD+Shift+2           Move active window to workspace 2
MOD+Shift+3           Move active window to workspace 3
MOD+Shift+4           Move active window to workspace 4
MOD+Shift+5           Move active window to workspace 5
MOD+Shift+6           Move active window to workspace 6
MOD+Shift+7           Move active window to workspace 7
MOD+Shift+8           Move active window to workspace 8
MOD+Shift+9           Move active window to workspace 9
MOD+Shift+Right       Move active window to workspace right
MOD+Shift+Left        Move active window to workspace left
MOD+Right             Change active window focus right (next)
MOD+Tab               Change active window focus right (next)
MOD+Left              Change active window focus left (previous)
MOD+Shift+Tab         Change active window focus left (previous)
Shift+Right           Resize (grow) master area wider
Shift+Left            Resize (shrink) master area narrower
MOD+Shift+space       Toggle active window floating
Control+Shift+Right   Resize active floating window width larger
Control+Shift+Left    Resize active floating window width smaller
Control+Shift+Up      Resize active floating window height smaller
Control+Shift+Down    Resize active floating window height larger
MOD+Control+Right     Move active floating window right
MOD+Control+Left      Move active floating window left
MOD+Control+Up        Move active floating window up
MOD+Control+Down      Move active floating window down
MOD+button1           Move active floating window
MOD+Shift+m           Maximize and unmaximize active window
MOD+Shift+f           Toggle active window fullscreen
MOD+o                 Show list of open windows
MOD+Shift+q           Close active window
MOD+space             Cycle layout type (vertical, horizontal, max, floating)
Print                 Screenshot entire desktop
Shift+Print           Screenshot selected area
MOD+Print             Screenshot active window
MOD+Control+equal     Disable dpms to keep session awake (like Caffeine)
MOD+Control+minus     Re-enable dpms to resume screen blanking and locking
MOD+x                 Lock screen
MOD+p                 Power menu (lock, logout, reboot, shutdown)
```
