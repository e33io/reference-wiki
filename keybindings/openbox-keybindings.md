# Openbox Keybindings

```
Modifier keys:

W         Windows (Super) key
A         Alt key
C         Control key
S         Shift key
Return    Enter key
space     Space bar
Left      Left arrow key
Right     Right arrow key
Up        Up arrow key
Down      Down arrow key
Tab       Tab key
equal     Equal key
minus     Minus key
Print     PrtSc (Print Screen) key
button1   Left mouse button
button3   Right mouse button
```
```
Custom keybindings:

W+d          Program launcher (app menu)
W+Return     Launch terminal
W+S+Return   Launch file manager
W+b          Launch web browser
W+1          Switch to workspace 1
W+2          Switch to workspace 2
W+3          Switch to workspace 3
W+4          Switch to workspace 4
W+5          Switch to workspace 5
W+6          Switch to workspace 6
C+A+Right    Switch to next workspace
C+A+Left     Switch to previous workspace
W+S+1        Send active window to workspace 1
W+S+2        Send active window to workspace 2
W+S+3        Send active window to workspace 3
W+S+4        Send active window to workspace 4
W+S+5        Send active window to workspace 5
W+S+6        Send active window to workspace 6
W+S+Right    Send active window to next workspace
W+S+Left     Send active window to previous workspace
A+Tab        Change active window focus to next window
W+A+Right    Change active window focus to next window
A+S+Tab      Change active window focus to previous window
W+A+Left     Change active window focus to previous window
W+S+f        Toggle active window fullscreen
W+S+m        Maximize and unmaximize active window
W+C+Right    Move active window right
W+C+Left     Move active window left
W+C+Up       Move active window up
W+C+Down     Move active window down
C+S+Right    Resize active window width larger
C+S+Left     Resize active window width smaller
C+S+Up       Resize active window height smaller
C+S+Down     Resize active window height larger
W+S+space    Move active window to center of screen
W+Right      Tile active window right
W+Left       Tile active window left
W+Up         Tile active window up
W+Down       Tile active window down
A+space      Show active window's menu
W+o          Show list of open windows
W+S+q        Close active window
Print        Screenshot entire desktop
S+Print      Screenshot selected area
W+Print      Screenshot active window
W+C+equal    Disable dpms to keep session awake (like Caffeine)
W+C+minus    Re-enable dpms to resume screen blanking and locking
W+x          Lock screen
W+p          Power menu (lock, logout, reboot, shutdown)

W+button1 in the window and drag        Move active window
W+button3 in the window and drag        Resize active window
button1 on the window border and drag   Resize active window
```
